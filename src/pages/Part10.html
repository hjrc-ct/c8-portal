Before you leave...

Let's do some clean up at the conclusion of this program.

Delete namespace


kubectl delete namespace $UNAMESPACE

Deleting the namespace will release resources such as persistent volume/disk(s), pods, secrets etc.
You may now close the apps such as Project Lens, VS Code and others.
As a best practice, please make sure to delete the token created in your Docker account.

TIP: It is a security best practice to clear cache at the conclusion of your labs exercise.
