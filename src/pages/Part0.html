Welcome !

C8-Labs is a hands-on learning platform for Camunda 8 Self-Managed on Kubernetes. In this self-paced program, you'll build and operate a complete Camunda 8 environment in a dedicated namespace within a Google Kubernetes Engine (GKE) cluster through guided labs. Each exercise includes detailed instructions, validation steps, and real-world scenarios to help you gain practical experience. Whether you're new to Camunda 8 or preparing for real-world deployments, C8-Labs provides a structured environment to learn by doing.

What You'll Learn / Do

By the end of the labs, you'll be able to:

  • Deploy Camunda 8 on Kubernetes/GCP
  • Understand the Camunda 8 architecture
  • Deploy and operate Zeebe Orchestration, Operate, Tasklist, Identity, Optimize, and Connectors
  • Inside view of Kubernetes objects and Topology
  • Manage Secrets and ConfigMaps
  • Configure storage and persistent volumes
  • Monitor workloads using Prometheus and Grafana
  • Troubleshoot deployment issues
  • Scale and manage Camunda components
  • Clean up cloud resources after completing the labs
