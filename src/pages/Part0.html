Welcome to C8-Labs : Managed by Community Chapter Bengaluru !

This self‑paced program is designed for practitioners working with modern cloud‑native architectures who want practical experience running Camunda on Kubernetes.

C8-Labs is a hands-on learning platform for Camunda 8 on Kubernetes. Instead of just watching videos, you'll build and operate a complete Camunda 8 environment in your own Google Kubernetes Engine (GKE) cluster through guided labs. Each exercise includes detailed instructions, automated validation, and real-world scenarios to help you gain practical experience. Whether you're new to Camunda 8 or preparing for production deployments, C8-Labs provides a structured environment to learn by doing.

What You'll Learn

By the end of the labs, you'll be able to:

  • Deploy Camunda 8 on Kubernetes/GCP
  • Understand the Camunda 8 architecture
  • Deploy Zeebe Orchestration, Operate, Tasklist, Identity, Optimize, and Connectors
  • Configure Ingress and HTTPS
  • Manage Secrets and ConfigMaps
  • Configure storage and persistent volumes
  • Monitor workloads using Prometheus and Grafana
  • Troubleshoot deployment issues
  • Scale and manage Camunda components
  • Clean up cloud resources after completing the labs
